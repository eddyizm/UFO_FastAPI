About the site

Built this site to experient with FastAPI, a newerish Python web framework. Of course, I also used this opportunity to grab some data from kaggle, a UFO dataset that came in the form of a csv. I thought this would make a great and fun data set.

Little did I know that the site was taken and enhanced from another data science site, link here, which seems to be the original person who scraped the website. Nuforc, the actual site that has been taking the time to receive and process these reports for 30 years here.

At this point, I still decided it would be fun to use this dataset to learn the frameworks above. I hope that the original website owners enjoy this fun mock modernized layout.
